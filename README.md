# s2u2m toolkit

## 后端

- Java 17
- Spring Boot 3.x
- Spring Security
- Maven
- google-java-formatter
- 

## 前端

- react，前端组建框架
- remix，前端全栈框架，提供路由，服务端渲染等功能
- eslint/prettier，对代码规范进行检测和格式化
- vite，构建工具
- vitest，前端单元测试

## Release
